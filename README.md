# introduce project
  
The project named **Feane** is a fast food web application, which is based on **Node.js**.  
Customers can use it to order food in the restaurant, place take-away orders, or book meals in advance.  
They can also track their orders directly through the web application.  

In addition, staff members can place orders on behalf of elderly customers or anyone who cannot use the web application in the restaurant for any reason.

---

## dependencies

**Languages:** TypeScript, HTML, CSS
**Environment:** NodeJS
**Environment Variable:** This project uses [dotenv](https://www.npmjs.com/package/dotenv) to load environment variables from a `.env` file into `process.env`
**Tools:** Git/Github, Visual Studio Code, Windows

### Frontend

- Framework/Library UI: NextJS, Tailwindcss, Shadcn UI
- Library: axios, zod

### Backend

- Framework/Library: NestJS
- ORM: Prisma
- Database: PostgreSQL
